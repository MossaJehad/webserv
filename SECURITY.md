# Security Policy

## Supported Versions

Security updates are applied to the latest stable version of this project.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |
| Older versions | ❌ |

Only the most recent version receives security fixes.  
Please make sure you are using the latest version before reporting issues.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

### How to Report

- Do **not** open a public issue for security vulnerabilities.
- Contact the maintainers privately.
- Include:
  - A clear description of the vulnerability
  - Steps to reproduce
  - Potential impact
  - Any suggested mitigation (if available)

### Response Process

- Reports will be reviewed as soon as possible.
- If confirmed, a fix will be prepared and released.
- Responsible disclosure is appreciated.
- Public discussion should happen only after the issue is resolved.

Thank you for helping keep this project secure.
